Justify-content

A propriedade CSS justify-content é usada em conjunto com o modelo de layout flexbox para controlar o alinhamento dos elementos ao longo do eixo principal (eixo de fluxo) dentro de um contêiner flexível. Ela define como os itens flexíveis são posicionados horizontalmente dentro do contêiner.

A propriedade justify-content afeta todos os itens dentro do contêiner flexível. No entanto, é possível substituir o alinhamento individual de um item usando a propriedade justify-self aplicada diretamente ao item.

Essa propriedade é útil para controlar a posição horizontal dos itens flexíveis dentro de um contêiner, permitindo criar layouts flexíveis e responsivos. Ela oferece diversas opções para distribuir, alinhar e espaçar os itens ao longo do eixo principal.

A propriedade justify-content aceita os seguintes valores:

Justify-content: flex-start;

flex-start (valor padrão): Os itens são alinhados no início do contêiner ao longo do eixo principal. Eles estarão à esquerda, se a direção principal for horizontal, ou no topo, se a direção principal for vertical.

Para que isso funcione, usamos as propriedades assim:
display: flex;
justify-content: flex-start;

Justify-content: flex-end;

flex-end: Os itens são alinhados no final do contêiner ao longo do eixo principal. Eles estarão à direita, se a direção principal for horizontal, ou na parte inferior, se a direção principal for vertical.

Para que isso funcione, usamos as propriedades assim:
display: flex;
justify-content: flex-end;

Justify-content: center;

center: Os itens são centralizados horizontalmente dentro do contêiner ao longo do eixo principal.

Para que isso funcione, usamos as propriedades assim:
display: flex;
justify-content: center;

Justify-content: space-around;

space-around: Os itens são distribuídos igualmente ao longo do eixo principal, com espaços em branco antes, entre e após eles. Ou seja, há espaço igual antes do primeiro item e após o último item, e os espaços entre os itens são iguais.
Porém, está propriedade divide os espaços que estão entre cada elemento, ou seja, ele adiciona automaticamente uma margem para cada elemento, logo, o elemento 01 estará com um margin-left e um margin-right no mesmo elemento. após isso, o elemento que está ao centro dos objetos, estará com uma distância maior de margin-left do elemento 01 e margin-rigth do elemento 02 dos objetos que estarão nas pontas do espaço disponibilizado.

Para que isso funcione, usamos as propriedades assim:
display: flex;
justify-content: space-around;

Justify-content: space-between;

space-between: Os itens são distribuídos igualmente ao longo do eixo principal, com espaços em branco entre eles. O primeiro item é alinhado ao início do contêiner e o último item é alinhado ao final.

Para que isso funcione, usamos as propriedades assim:
display: flex;
justify-content: space-between;

Justify-content: space-evenly;

space-evenly: Os itens são distribuídos igualmente ao longo do eixo principal, com espaços em branco antes, entre e após eles. Os espaços em branco são distribuídos de forma igualitária, incluindo o espaço antes do primeiro item e após o último item.

Para que isso funcione, usamos as propriedades assim:
display: flex;
justify-content: space-evenly;
