Align-items

A propriedade CSS align-items é usada em conjunto com o modelo de layout flexbox para controlar o alinhamento dos elementos ao longo do eixo transversal (eixo perpendicular) dentro de um contêiner flexível. Ela define como os itens flexíveis são posicionados verticalmente dentro do contêiner.

A propriedade align-items afeta todos os itens dentro do contêiner flexível. No entanto, é possível substituir o alinhamento individual de um item usando a propriedade align-self aplicada diretamente ao item. Essa propriedade é útil para controlar a posição vertical dos itens flexíveis dentro de um contêiner, permitindo criar layouts flexíveis e responsivos.

A propriedade align-items aceita os seguintes valores:

align-items: flex-start;

flex-start: Os itens são alinhados no início do contêiner ao longo do eixo transversal. Eles estarão no topo, se a direção principal for horizontal, ou à esquerda, se a direção principal for vertical.
Para que nossos elementos fique direcionanos horizontalmente junto ao align-itens, precisamos usar uma propriedade que nos forneça controle sobre o posicionamento horizontal, como a propriedade justify-content.

Para que isso funcione, usamos as propriedades assim:
display: flex;
justify-content: center;
align-items: flex-start;

align-items: flex-end;

flex-end: Os itens são alinhados no final do contêiner ao longo do eixo transversal. Eles estarão na parte inferior, se a direção principal for horizontal, ou à direita, se a direção principal for vertical.

Para que isso funcione, usamos as propriedades assim:
display: flex;
justify-content: center;
align-items: flex-end;

align-items: center;

center: Os itens são centralizados verticalmente dentro do contêiner ao longo do eixo transversal.

Para que isso funcione, usamos as propriedades assim:
display: flex;
justify-content: center;
align-items: center;

align-items: stretch;

stretch (valor padrão): Os itens são esticados para preencher o contêiner verticalmente. Se necessário, eles serão redimensionados para ter a mesma altura.

Para que isso funcione, usamos as propriedades assim:
display: flex;
justify-content: center;
align-items: stretch;

align-items: baseline;

baseline: Os itens são alinhados com suas linhas de base. A linha de base é determinada pelo conteúdo do item, como o texto dentro de um elemento de texto.

justify-content: center;
