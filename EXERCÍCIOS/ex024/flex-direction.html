Flex-direction

'flex-direction' é uma propriedade CSS que define a direção do fluxo dos elementos dentro de um contêiner flexível. Ela é usada em conjunção com o modelo de layout flexbox para controlar o posicionamento e a ordem dos itens em um contêiner flexível.

Esses valores afetam a direção em que os elementos são posicionados no eixo principal do contêiner flexível. O eixo principal é determinado pela direção do fluxo definida pela propriedade 'flex-direction'. O eixo transversal é perpendicular ao eixo principal.

Além disso, a propriedade flex-direction também influencia a ordem em que os elementos são renderizados. Por padrão, os elementos são renderizados na ordem em que aparecem no código HTML, mas a ordem pode ser alterada usando propriedades como 'order'.

Em resumo, a propriedade flex-direction permite controlar a direção do fluxo e a ordem dos elementos em um contêiner flexível, facilitando a criação de layouts flexíveis e responsivos.

Existem quatro valores possíveis para a propriedade flex-direction:

flex-direction: row;

row: Os elementos são dispostos horizontalmente da esquerda para a direita. Este é o valor padrão se nenhum valor for especificado.

Para que isso funcione, usamos as propriedades assim:
Display: flex;
flex-direction: row;
